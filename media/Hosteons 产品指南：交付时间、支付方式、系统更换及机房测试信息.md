## 操作系统更换指南

Hosteons VPS 提供灵活的操作系统更换功能：

1. **支持的操作系统**：
   - Linux 发行版：CentOS、Fedora、Debian、Ubuntu
   - Windows 系统：2008/2012（vps3及以上套餐支持）
   - FreeBSD（需通过 VNC 安装）

2. **更换方式**：
   - 通过 VPS 控制面板直接更换
   - Linux VPS 可更换为 Windows 系统（需套餐支持）
